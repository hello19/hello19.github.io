    先用命令查询一下brew search python3

    可以查询到，直接安装brew install python3

    安装完成，输入python3查看信息，然后输入brew info python3查看当前系统下的python3信息

    到这一步已经安装完成，在系统中同时可以使用两个版本的python了

生成SSH Key

SSH是一个专为远程登录会话和其他网络服务提供安全性的协议。默认状态下SSH链接是需要密码认证的，可以通过添加系统认证（即公钥-私钥）的修改，修改后系统间切换可以避免密码输入和SSH认证。

  • 查看本地是否存在SSH KEY

    ls -al ~/.ssh

    No such file or directory证明本地没有生成的SSH Key

    id_rsa id_rsa.pub出现这两个文件则证明已经存在SSH KEY

  • 生成秘钥

    ssh-keygen -t rsa

    -t 的意思是选择kye的type。分别有 RSA 和 DSA 两种

    第一次确认key的生成位置，第二次是密码，第三次是确认密码，建议三次回车直接生成

    按照默认，私钥被放在了~/.ssh/id_rsa 这个文件里，而公钥被放在了 ~/.ssh/id_rsa.pub 这个文件里

  • 将秘钥加到服务器

    服务器添加的秘钥都在~/.ssh/authorized_keys这里面，把id_rsa.pub这里面的内容复制粘贴到authorized_keys里就OK了

    或者在复制完id_rsa.pub里的内容后执行命令：cat id_dsa.pub >> ~/.ssh/authorized_keys
